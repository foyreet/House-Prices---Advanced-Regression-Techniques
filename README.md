# House-Prices - Advanced-Regression-Techniques
https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/overview

Для предсказания использована мощная ансамблевая модель стекинга (StackingRegressor), объединяющая 6 базовых алгоритмов с финальной мета-моделью RidgeCV
Выполнил Feature Engineering: данные очищены от пропусков и слабых признаков, а целевая цена логарифмирована для нормализации
Результат по метрике **RMSE составил 0.14173**
